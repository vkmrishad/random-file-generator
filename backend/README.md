# random-file-generator(backend)
Random file generator (Backend)

## Environment and Package Management
Install Poetry

    $ pip install poetry
    or
    $ pip3 install poetry

Activate or Create Env

    $ poetry shell

Install Packages from Poetry

    $ poetry install

NB: We using virtualenv, install from `$ pip install -r requirements.txt`.

## Runserver

    $ python app.py

## API Endpoints
